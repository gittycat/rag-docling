About RAG System

What is this?

This is a local Retrieval Augmented Generation (RAG) system that allows you to search through your documents using natural language queries. It combines document retrieval with large language models to provide intelligent, context-aware answers.

Features

  • Natural language document search
  • Support for multiple file formats (txt, md, pdf, docx)
  • Local deployment with privacy-first design
  • Powered by Ollama for LLM capabilities
  • ChromaDB for efficient vector storage

Technology Stack

  • Frontend: FastAPI + Tailwind CSS
  • RAG Server: Python + FastAPI
  • Vector Database: ChromaDB
  • LLM: Ollama (local deployment)
  • Embeddings: nomic-embed-text (768 dimensions)
  • Document Processing: MarkItDown
